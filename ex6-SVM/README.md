# 支持向量机部分学习总结

## 支持向量机简述 

    支持向量机是监督学习算法中的一种，其主要是用在分类任务中，典型的使用场景有垃圾邮件的分类。
  
## 本章学习重点

* 理解支持向量机的数学原理及其和逻辑回归的不同之处。

* 理解支持向量机中核函数的数学原理，及其作用。

* 代码实现支持向量机。
